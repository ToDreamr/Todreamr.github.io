删除元素

指的是覆盖而不是的的确确的删除

C++ vector作为数组的封装

erase函数：删除：O(n)

双指针实现

```c++
class Solution {
public:
    int removeElement(vector<int>& nums, int val) {
         //定义快慢指针
        int fast = 0;//获取新元素
        int slow = 0;//获取更改位置
        for(fast = 0; fast < nums.size(); fast++){
            if(nums[fast] != val){
                nums[slow++]=nums[fast];
            }
        }
        return slow;
    }
};
```
